Welcome to Curaga!

This application is intended to be used in two ways

  • To create/upload packages using the curaga client library.
  • To download/install packages using curl

To create a package:

Download curaga and use --help

To download a package:

At a shell do "curl http://localhost:8000/package/ | /bin/sh". This will download the .zip, unzip it to ~/.curaga_packages, backup any files the package contains, and makes sym links to the files in the package

Reserved urls:

  • /
  • /upload
  • /package/
  • /auth/register
